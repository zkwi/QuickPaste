# QuickPaste v0.8.0 发布说明

v0.8.0 统一了设置、管理和图片预览的布局，并让 OCR 搜索和应用内更新更容易理解与使用。本版本以 [GitHub Pre-release](https://github.com/zkwi/QuickPaste/releases/tag/v0.8.0) 分发；真实机长循环仍保留为待验证项，因此不表述为稳定版已全面验收。

## 主要变化

- 常用设置前置：设置页顶部直接提供“管理剪贴板”和全局快捷键入口，正文、说明和按钮字体整体增大，紧凑窗口自动切为单列。
- 管理页整理：搜索、结果状态、新建片段、筛选和批量操作分组排布；操作区按宽度换行，不再重叠。
- 图片预览收紧：标题、来源和时间合并为紧凑元信息，移除重复的 `IMAGE` 徽标，让图片优先占用首屏。
- OCR 搜索明确：快速面板与管理页搜索框明确支持 OCR 文字；本地识别结果继续进入 FTS 索引并显示 OCR 命中标记。
- 更新自动安装：发现新版本后点击“下载安装”，应用会依次下载、校验、保存当前历史并静默启动 NSIS 安装，安装完成后重新打开。
- 减少干扰：主标题栏不再放置低频暂停按钮；备份区域不再显示数据库未加密提示；更新界面和发布页不再强调签名状态。

## 隐私与资源边界

OCR 继续只使用 Windows 已安装语言包，不下载模型，也不上传图片、识别文字或搜索词。剪贴板正文、图片和 OCR 结果只保存在本机 SQLite 中。更新检查只访问 `zkwi/QuickPaste` 的公开 GitHub Release，下载后核对 GitHub 提供的 SHA-256 摘要。

## 验收状态

本轮回归覆盖设置主入口、快捷键录制、OCR 查询、图片首屏预览、批量工具栏换行、中文输入法组合态、无提示备份和单次点击更新安装。完整命令、最终测试数量、安装包哈希与未完成证据记录在 [v0.8.0 验证报告](https://github.com/zkwi/QuickPaste/blob/main/docs/releases/v0.8.0-verification.md)。

真实机快捷键时延、10,000 次回贴、100,000 次捕获、多屏全 DPI、管理员窗口、Windows OCR 语言包、安装/覆盖安装/卸载和网络抽查继续保持 **pending real-machine**；自动化通过不能替代这些证据。

## 已知限制

- Windows OCR 的语言与质量取决于系统已安装语言包；不可用、过大或失败会显示对应状态。
- 自动回贴、管理员 helper、屏幕捕获排除和多显示器定位仍受目标应用及 Windows 行为影响。
- 仅构建 Windows x64 当前用户 NSIS，不提供 MSI；完整真实机器安装证据尚待补齐。
