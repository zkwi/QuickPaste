- 标准窗口快速面板实测 `768 × 548`，记录行高约 36 px，单行内容高约 18 px，页面无水平或垂直溢出。

- 图片记录不再重复展示尺寸；真正不同的标题和正文使用“标题 · 正文”同排显示。
- 浅色/深色切换、键盘可访问名称和中文界面正常；浏览器控制台错误为 0。

曾使用隔离 `paste-ordinary` profile 启动最终候选，marker、临时 app-data 与 WebView2 目录均正确创建；但准备执行候选到 Notepad 的真实双击回贴时桌面已锁定。遵守自动化安全边界未尝试绕过锁屏，因此本轮最终候选的外部普通权限目标回贴仍记为 **pending real-machine**，不把较早安装版的手工结果移作本候选证据。

## GitHub 发布与回下载核验

发布操作在不可变候选提交完成后执行；标签提交、Release/Asset ID、GitHub `asset.digest`、公开下载回读摘要与对应异步 CI 链接以本次发布任务最终记录为准。远端 CI 只作推送后的异步复核，不替代上述本地阻断门禁。

## 仍待真实机器验证

- 50 次预热 + 500 次快捷键到首帧采样，p95 ≤ 120 ms。
- 由独立普通权限目标实际读取 10,000 次粘贴内容，成功率 ≥ 99.5%。
- 由独立 writer/expected-ID ledger 执行 100,000 次捕获并与数据库/事件对账，漏记、异常重复和自身回流合计 < 0.1%。
- 100%、125%、150%、175%、200%、225%、250% DPI，多显示器不同缩放、负坐标和四任务栏边缘无裁切错位。
- Explorer/Office/浏览器/管理员目标、中文输入法、复杂 OLE 对象、安装/覆盖安装/卸载，以及真实机网络抽查。
