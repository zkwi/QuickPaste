## 界面回归

使用真实 Chromium 页面在浅色、深色和 `720 × 620` 紧凑视口检查了快速面板、管理页、设置页和图片详情：

- 快速列表在标准窗口显示 10 条记录，来源应用与相对时间上下排列，正文没有被操作区遮挡。
- 管理页仅保留全文搜索与类型筛选，未选择记录时不显示批量动作；列表来源与时间上下排列。
- 设置页卡片在宽窗口保持两列，在紧凑宽度按既有断点回落；主页面与设置页面演示截图已写入 README。
- 图片详情使用同一内容区容纳图片与 OCR 文字，并在窄窗口回落为单列。
- 深色主题和紧凑窗口没有可见溢出、遮挡或横向滚动；浏览器控制台无 error。

README 图片由仓库内置演示数据生成，不含真实剪贴板、账户、路径或设备信息。隐私门禁只对白名单中的两张固定产品预览图放行，其他 raster 图片仍会被拒绝。
