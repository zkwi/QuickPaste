# QuickPaste v0.17.0 验证报告

验证日期：2026-07-21

本报告记录 v0.17.0 发布候选的本地自动化、界面、安全审计和产物验证证据。真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 自动化与安全审计

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，43 项治理测试通过。
- 前端测试：34 个测试文件、715 项测试通过；针对全量并发下的测试初始化时序修正后，额外连续 3 轮全量测试均为 715/715 通过；TypeScript 检查与 Vite 生产构建通过。
- Rust 格式、Clippy（`-D warnings`）及测试：316 项测试通过。
- npm 全依赖与 production-only advisories 审计：0 项已知漏洞；RustSec 数据库审计：0 项已知漏洞。
- Tauri CSP 保持仅允许本地脚本，悬停文本继续使用 Vue 文本插值，未引入 `v-html`、远程脚本、额外窗口权限或新的运行时依赖。
- `npm run build:windows`：通过，只生成 v0.17.0 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 界面检查范围

- Playwright 在 640×440 紧凑窗口完成浅色/深色文本与图片悬停预览检查；图片卡片约 319×228px，内容区约 298px。
- 悬停后键盘选中项保持不变，搜索框焦点不被夺走；浏览器控制台无错误。
- 自动化覆盖 200ms 延迟、离开即关闭、原生完整图片载荷、超长文本有界挂载、Alt 徽标显示规则和中文输入法组合态。
