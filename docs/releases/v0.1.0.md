# QuickPaste v0.1.0（未签名预发布）

这是闪电剪贴板 QuickPaste 的首个公开预发布版本，面向 Windows x64。

## 主要功能

- 使用 `Ctrl + Shift + V` 在原输入位置附近唤起紧凑面板，并在选中后自动回贴原窗口。
- 支持文本、代码、链接与图片剪贴板历史，以及中文原文、全拼和拼音首字母搜索。
- 搜索词高亮，重新唤起快捷面板时清空上次搜索，键盘与鼠标操作保持一致。
- 动态读取来源应用图标，统一右侧应用信息对齐，并改进高信息密度布局。
- 设置页和托盘支持检查 GitHub Release；用户确认后下载并核对 SHA-256，再启动 NSIS 更新。
- 同内容再次复制会刷新来源和时间，应用自己的复制/回贴不会污染历史；图片保留原始分辨率。
- 历史数据仅保存在本机 SQLite；无云同步、无遥测。
- 主程序保持普通用户权限，需要向管理员窗口粘贴时使用一次性授权 helper。
