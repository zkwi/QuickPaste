# QuickPaste v0.21.1 发布说明

v0.21.1 修复快速面板在第三方输入法参与窗口焦点切换时可能间歇性闪退，并收紧无选中项与粘贴失败时的焦点恢复。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机长期矩阵仍保留为待验证项，因此不表述为已经全面验收。

## 主要修复

- 快速面板定位不再调用可能引入第三方 MSAA/UIA 代理的可访问性接口，仅使用 Windows GUI 线程与插入点信息；无法取得可靠插入点时回退到鼠标位置，避免第三方组件的堆损坏拖垮 QuickPaste。
- 原生粘贴前先结束搜索框输入法组合态并释放 WebView 焦点，避免隐藏窗口与输入法焦点销毁重叠；仅复制或粘贴失败后恢复原焦点。
- 无搜索结果或无选中记录时按 `Enter` 不触发粘贴；新增无选中项、粘贴前失焦、失败后恢复焦点及禁止进程内 MSAA 查询的回归测试。

## 数据、安全与兼容性

- 本版本不改变 SQLite schema、历史数据目录、剪贴板载荷格式、设置 schema 或管理员授权协议，无需迁移。
- 目标窗口、剪贴板版本、物理修饰键、helper 授权时限和管理员权限校验保持不变。
- 部分无法通过系统 GUI 线程取得精确插入点的应用，快速面板可能改为靠近鼠标位置显示；搜索、选择与粘贴流程不变。
- 主程序继续保持普通用户权限；管理员窗口仍只通过现有一次性 UAC helper 粘贴。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI、MSIX 或 APPX。
