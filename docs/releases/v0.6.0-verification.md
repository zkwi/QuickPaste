## 10,000 条历史合成基准

命令：

```powershell
cargo run --release --locked --manifest-path src-tauri/Cargo.toml --example history_bench
```

口径为 schema 9、固定种子 10,000 行、100 次预热和 1,000 个混合查询。最终结果：

| 指标 | 结果 |
| --- | ---: |
| 数据准备 | 1,782.923 ms |
| p50 | 7.105 ms |
| p95 | 26.795 ms |
| p99 | 48.318 ms |
| 阈值 | p95 ≤ 50.000 ms |

该结果属于 synthetic benchmark，只证明受控 SQLite 查询负载，不代表快捷键首帧、真实应用粘贴或捕获完整性。

- 50 次预热 + 500 次快捷键到首帧采样，p95 ≤ 120 ms。
- 由独立普通权限目标实际读取 10,000 次粘贴内容，成功率 ≥ 99.5%。
- 由独立 writer/expected-ID ledger 执行 100,000 次捕获并与数据库/事件对账，漏记、异常重复和自身回流合计 < 0.1%。
- 100%、125%、150%、175%、200%、225%、250% DPI，多显示器不同缩放、负坐标和四任务栏边缘无裁切错位。
- Explorer/Office/浏览器/管理员目标、中文输入法、Windows 已安装语言包 OCR、安装/覆盖安装/卸载，以及真实机网络抽查。
