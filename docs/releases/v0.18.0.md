# QuickPaste v0.18.0 发布说明

v0.18.0 集中完成规划中的高优先级无障碍、搜索帮助、快速面板密度、悬浮预览与供应链治理改进。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机长循环仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- 搜索框现在使用标准组合框语义，来源建议拥有稳定列表与活动选项关系；建议关闭时不再错误指向剪贴板结果列表。
- 搜索框新增键盘可访问的帮助入口，集中说明 `@来源`、`;永久片段`、空格预览、Enter 粘贴与 Alt + 1…0 直达粘贴。
- 管理页使用普通列表与原生复选框表达批量选择，保留键盘多选、Shift 连选、跨页全选和现有批量操作。
- 标准与紧凑快速面板分别使用 44px 与 40px 行高，至少完整显示 8 条与 6 条记录。
- 文本悬浮预览只在摘要确实被截断时出现，并采用 400ms 意图延迟；滚动、按键、指针按下、搜索或筛选变化及窗口失焦会立即关闭。
- GitHub Actions 全部固定到完整提交 SHA，并增加 npm/Cargo/GitHub Actions Dependabot 周更与定期 npm、RustSec 安全审计。

## 数据与兼容性

- 本版本不改变 SQLite schema、数据目录或历史保留规则，无需迁移。
- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹。复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI。

## 尚待真实机验证
