# QuickPaste v0.19.1 发布说明

v0.19.1 修复拼音或首字母通过原生搜索索引命中时，快速面板只显示短标题、未继续利用两行正文摘要的问题。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机矩阵仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- 仅索引命中的文本、代码等记录现在复用普通状态下的统一摘要规则，在保留“索引命中”提示的同时展示最多两行有效正文。
- 图片、链接和文件仍按各自类型生成摘要；搜索不会为显示结果额外加载完整剪贴板载荷，继续使用原生历史查询返回的有界摘要。
- 直接文字搜索高亮、OCR 命中提示、键盘选择、双击和 Enter 粘贴行为保持不变。

## 数据与兼容性

- 本版本不改变 SQLite schema、数据目录、历史保留规则、捕获协议或粘贴协议，无需迁移。
- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹。复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI、MSIX 或 APPX。
- 当前发布二进制未进行 Authenticode 签名。
