# QuickPaste v0.19.2 发布说明

v0.19.2 修复多语言搜索结果中的“拼音命中”“索引命中”和 “OCR 命中”徽标可能换行并撑高条目的问题，并补齐 Windows 强制色模式下的可辨识边框。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机矩阵仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- 搜索来源徽标固定保持单行、不可收缩，不再因空间竞争、长英文译文或窄窗口变为竖排文字并增加记录行高。
- 快速面板标准与紧凑模式仍保持 44px 与 40px 的既定行高；管理页仍保持 54px 的紧凑行高。
- Windows 强制色模式为拼音与索引命中徽标使用系统色边框，避免高对比度主题下状态轮廓消失。
- 中文、英文、长英文连续词、阿拉伯语、日文、韩文和 Emoji 内容均完成定向布局检查；搜索、选择与粘贴流程不变。

## 数据与兼容性

- 本版本不改变 SQLite schema、数据目录、历史保留规则、捕获协议或粘贴协议，无需迁移。
- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹。复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI、MSIX 或 APPX。
- 当前发布二进制未进行 Authenticode 签名。
