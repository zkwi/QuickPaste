# QuickPaste v0.17.1 验证报告

验证日期：2026-07-22

本报告记录 v0.17.1 发布候选的本地自动化、安全审计和产物验证证据。按维护者要求，本次不使用 Computer Use；真实应用完整交互矩阵、真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 缺陷回归

- 使用脱敏的 Chrome 风格多段 HTML（含超链接、空段落和 Unicode 文本）验证 `CF_UNICODETEXT` 与 `HTML Format` 写入后可完整回读。
- 写入后由独立线程占用系统剪贴板 300ms，模拟浏览器、远程剪贴板或其他读取器竞争；成功写入仍返回确定的剪贴板序列，不会降级为仅复制。
- QuickPaste 内部写入标记会让监听读取结果进入 `Ignored`，同时保留完整格式包供粘贴前校验；写入失败仍取消自写抑制状态。

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，43 项治理测试通过。
- 前端测试：34 个测试文件、715 项测试通过；TypeScript 检查与 Vite 生产构建通过。
- Rust 格式、Clippy（`-D warnings`）及测试：317 项测试通过。
- `npm run build:windows`：通过，只生成 v0.17.1 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。
- GitHub Actions [CI 运行 #29882932520](https://github.com/zkwi/QuickPaste/actions/runs/29882932520)：发布时为 `in_progress`，作为推送后的异步复核，不替代或阻塞本地发布门禁。
- Tauri CSP 与运行时权限未变；未引入新依赖、远程脚本、额外窗口权限或数据库 schema 迁移。
- 发布产物只允许 Windows x64 NSIS 与绿色版 ZIP，不生成 MSI、MSIX 或 APPX。
