# QuickPaste v0.13.0 验证报告

验证日期：2026-07-20

本报告记录 v0.13.0 发布候选在本地 Windows 构建环境中的自动化、界面和产物验证证据。真实设备长时间运行及安装生命周期仍单独列为待验证项。

## 自动化验证

- `npm ci`：通过，共安装并审计 140 个包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过。
- 前端测试：31 个测试文件、670 项测试通过。
- TypeScript 检查与 Vite 生产构建：通过。
- Rust 格式、Clippy（`-D warnings`）及测试：290 项测试通过。
- `npm run build:windows`：通过，只生成 v0.13.0 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 界面与交互验证

使用真实浏览器页面检查了首次引导和练习提示：

- 800×580 标准窗口与 768×460 紧凑窗口。
- 浅色与深色主题。
- 首次引导在窗口中居中，最后一步的“直接开始”和“添加示例并练习”按钮无挤压或重叠。
- 练习提示保持单行紧凑高度，欢迎记录默认选中，历史列表仍是视觉主体。
- 中文界面、键盘焦点与可访问名称均可从页面快照读取；浏览器控制台无 warning 或 error。

自动化回归另行覆盖：

- 首次引导调用原生居中，完成引导后的日常启动不重复居中。
- 标题栏非交互区域触发原生拖动，窗口按钮不会误触拖动。
- 可选欢迎记录使用稳定 ID、空删除列表和独立 upsert，不替换已有历史，也不调用系统剪贴板复制或粘贴接口。
- 设置 schema 6 保存练习状态；托盘点击与全局快捷键调用同一前台目标激活路径。
