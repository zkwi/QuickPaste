# QuickPaste v0.21.0 验证报告

验证日期：2026-07-24

本报告记录 v0.21.0 剪贴板生命周期审计、双语 README 治理、自动化、安全审计、Windows 构建与 GitHub 发布证据。整个任务不使用 Computer Use；长期运行、完整 DPI 与外部应用组合矩阵继续单独列为 `pending real-machine`。

## 根因与缺陷回归

- v0.20.2 已证明格式包在显式 Windows 剪贴板 guard 存活时读取序列，会把关闭后的稳定状态误判为外部变化。本轮按同一资源生命周期审计所有 `seq_num`、`Clipboard::new`、格式包读写和粘贴授权路径。
- 审计发现敏感历史标记的提前返回分支仍在 guard 存活时读取终态序列。新增测试先因缺少“释放后观察”边界而编译失败，证明生产代码没有表达该契约。
- 最小修复把该早退路径统一为先 `drop(guard)`、再读取序列；相同序列返回 `Ignored`，序列变化返回 `Retryable`，没有放宽去重或粘贴授权。
- 定向 GREEN 覆盖析构顺序、文件元数据释放顺序、真实 Windows Chrome 风格多段 HTML 往返、内部格式包一次性抑制、旧纯文本写入兼容和管理员 helper 精确序列绑定。
