# QuickPaste v0.12.0 验证报告

验证日期：2026-07-20

本报告记录 v0.12.0 发布候选在本地 Windows 构建环境中的自动化、界面和产物验证证据。真实设备长时间运行及安装生命周期仍单独列为待验证项。

## 自动化验证

- `npm ci`：通过，共安装并审计 140 个包。
- `npm run check`：通过。
  - 仓库治理测试：38 项通过。
  - 前端测试：31 个测试文件、664 项测试通过。
  - TypeScript 检查与 Vite 生产构建：通过。
  - Rust 格式、Clippy（`-D warnings`）及测试：289 项测试通过。
- `npm run build:windows`：通过，只生成 NSIS 安装包，未生成 MSI、MSIX 或 APPX。
- 版本、隐私和第三方许可检查：通过；npm 许可清单含 28 个包，Rust 许可清单含 309 个 crate。

## 界面验证

使用真实浏览器页面检查了快速面板、图片详情、管理页和设置页：

- 标准窗口与 640×440 紧凑窗口。
- 浅色与深色主题。
- 来源应用图标、名称与时间的垂直对齐。
- 管理页筛选条在窄内容区不逐字断行、不与统计或新建按钮重叠，并可横向滚动。
- 设置页常用入口、快捷键、更新操作和存储维护区域在窄内容区无文字或控件重叠。
- 图片详情关闭 OCR 时，紧凑窗口主图高度约为 152 px；展开 OCR 后约为 130 px，页面无水平溢出。
- README 的快速面板和设置页截图使用合成示例内容重新生成，不包含真实剪贴板数据。
