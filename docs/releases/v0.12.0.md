# QuickPaste v0.12.0 发布说明

v0.12.0 集中优化快速面板、图片详情、管理工具栏和设置页在真实内容宽度下的排版。本版本以标准 [GitHub Release](https://github.com/zkwi/QuickPaste/releases/tag/v0.12.0) 分发，不标记为 Pre-release；真实机长循环仍保留为待验证项，因此不表述为已经全面验收。

## 重点变化

- 快速面板移除底部状态栏。目标应用和管理入口移到标题栏，历史列表与预览区获得更多垂直空间。
- 图片标题并入返回栏，不再为图片显示对象格式省略提示；OCR 文字默认折叠、可按需展开，主图在标准窗口和 640×440 紧凑窗口中都保持为视觉主体。
- 来源应用图标、名称和时间使用明确行高与居中约束，快速列表和管理列表不再显得上下偏斜。
- 管理页搜索和主操作位于首行，类型筛选独立位于第二行并禁止逐字压缩，窄内容区可安全横向滚动。
- 设置页使用稳定单列轨道；常用入口、快捷键、更新操作和存储维护卡片按实际内容宽度自动换行，避免文字与控件重叠。
- README 的快速面板和设置页截图已按新版界面重新生成。

## 隐私与兼容性

- OCR 仍只使用 Windows 本机能力，不下载模型、不上传图片或识别文字。
- 数据库格式没有变化，不需要迁移历史数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI。

## 尚待真实机验证

- Windows 10/11 的覆盖安装、卸载与重装。
- 100%–250% DPI、多显示器负坐标和任务栏边界。
- 长时间剪贴板监听、管理员目标回贴和 WebView2 缺失路径。

以上项目仍为 `pending real-machine`。维护者选择将 v0.12.0 作为标准 GitHub Release 分发，但这不改变上述验证状态。
