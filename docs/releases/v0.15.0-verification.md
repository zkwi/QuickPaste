# QuickPaste v0.15.0 验证报告

验证日期：2026-07-21

本报告记录 v0.15.0 发布候选的本地自动化、界面和产物验证证据。真实设备长时间运行及完整安装生命周期继续单独列为 `pending real-machine`。

## 自动化验证

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过。
- 前端测试：33 个测试文件、685 项测试通过。
- TypeScript 检查与 Vite 生产构建：通过。
- Rust 格式、Clippy（`-D warnings`）及测试：296 项测试通过。
- `npm run build:windows`：通过，只生成 v0.15.0 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 人工界面检查

- Playwright 真实 Chromium 渲染覆盖 640×440 浅色快速面板、640×440 深色设置页和 800×580 深色快速面板，未发现文字、列表、筛选或设置卡片重叠/裁切。
- `@微` 显示微信来源建议，Enter 选择后只保留微信结果并显示来源筛选标签；`;` 显示“永久片段”模式标签和对应空状态。
- 标题栏“面板常驻”和条目“固定内容”在图标、无障碍名称及 Tooltip 上可区分；设置页显示 `Ctrl + Shift + V` 的纯文本粘贴冲突提示。
- 浏览器控制台为 0 error。中文输入法组合态、OCR/二维码 IPC 与失焦状态机由自动化回归覆盖；真实 Tauri 窗口中的输入法和本地识别仍保留在下方真实机矩阵。
