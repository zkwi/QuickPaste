# QuickPaste v0.2.0（未签名预发布）

这是闪电剪贴板 QuickPaste 的 `0.2.0` Windows x64 预发布版本。

## 本轮改进

- 修复高 DPI 或较小有效视口下，引导页自动聚焦“下一步”后把品牌标题和“跳过”按钮滚出可视区的问题。
- 引导页在 `640 × 440` 等紧凑窗口中自动收紧标题栏、演示区、正文和底栏，三步内容均可完整显示。
- 正常支持尺寸不再出现多余滚动条；系统大字体、长翻译等极端溢出仍保留可滚动兜底。
- 引导遮罩铺满当前窗口，并统一普通窗口与最大化窗口的边界处理。

## 下载与校验

- 系统：Windows 10/11 x64
- 安装方式：NSIS 当前用户安装
- 文件：`QuickPaste_0.2.0_x64-setup.exe`
- 大小：4,176,450 字节
- SHA-256：`CCE1EA2134A5D333167DEABA9B45F9C385337CC993707A0761C284EBE8D004F4`
- Authenticode：未签名

此哈希只对应上述安装包。若后续代码、依赖或构建环境发生变化，必须重新构建并更新大小与 SHA-256；发布流程还需从 GitHub 下载入口复核资产摘要和大小。
