# QuickPaste v0.19.2 验证报告

验证日期：2026-07-22

本报告记录 v0.19.2 多语言搜索命中徽标布局修复的自动化、界面和候选产物验证证据。本次不使用 Computer Use；真实应用完整交互矩阵、真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 缺陷回归

- 构造中文、英文、长英文连续词、阿拉伯语 RTL、日文、韩文和 Emoji 内容，以及较长来源应用名称；同时覆盖拼音索引命中和 OCR 命中徽标。
- 修复前，徽标作为可收缩 flex 项会在空间竞争时逐字换行，浏览器的自动最小尺寸进而撑高整条记录；修复后徽标固定为不可收缩的单行内容。
- Windows 强制色模式下，修复前拼音与索引命中徽标没有系统色边框；修复后与 OCR 状态一样使用 `CanvasText` 边框。
- 两项修复均先加入失败回归测试，再进行最小 CSS 修改并验证转绿。

## 自动化与安全审计

- 多语言、样式、快速面板与设置定向测试：4 个测试文件、207 项测试通过。
- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，46 项治理测试通过。
- 前端测试：34 个测试文件、727 项测试通过；TypeScript 检查与 Vite 生产构建通过。
- `npm run check` 最终完整复核通过：Rust 格式、Clippy（`-D warnings`）及 317 项 Rust 测试全部通过。
- `npm audit --omit=dev --audit-level=high`（官方 npm registry）：通过，0 个漏洞。
- `cargo audit --file src-tauri/Cargo.lock`：退出码为 0，没有阻断性漏洞；报告 17 条允许的维护状态或 soundness 警告，来自 Tauri 的 GTK3/UNIC 间接依赖。QuickPaste 当前只交付 Windows，这些依赖不进入本次 Windows 可执行文件；警告仍由每周 RustSec 工作流持续监控。
- `npm run build:windows`：通过，只生成 v0.19.2 x64 NSIS 候选安装包，未生成 MSI、MSIX 或 APPX。

## 界面检查

- 使用 Playwright CLI 在 640×440 英文浅色界面验证快速面板：紧凑记录行高保持 40px，英文 `Index match` 徽标保持单行且不收缩，正文、徽标与来源元数据无重叠。
- 同一数据在管理页保持 54px 行高，搜索结果、来源信息和行内操作无重叠；方向键选择与焦点同步正常。
- 360×416 深色窄窗口压力检查中，所有搜索结果仍保持 40px 行高，徽标未换行且未与元数据重叠。
- Windows 强制色模式修复前后对比确认：拼音/索引徽标由无边框变为 1px 系统色实线边框，记录行高不变。
- 深浅主题控制台均为 0 个错误、0 个警告；中文输入法组合态由既有回归测试覆盖。
