# QuickPaste v0.19.0 发布说明

v0.19.0 提升快速面板与管理页的条目信息密度：快速面板在原有行高内默认显示两行，管理页不再重复展示由正文生成的标题。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机矩阵仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- 快速面板的摘要默认显示最多两行，标准与紧凑模式仍保持 44px 与 40px 行高，并继续满足至少完整显示 8 条与 6 条记录的密度门槛。
- 管理页将标题与正文整理为单一自适应摘要；重复标题只显示一次，独立标题和正文合并展示，图片、链接和多文件条目保留适合其类型的摘要。
- 搜索命中预览、高亮、OCR/拼音/缺失文件徽标、键盘选择和行内操作均保持原有行为。

## 数据与兼容性

- 本版本不改变 SQLite schema、数据目录、历史保留规则或粘贴协议，无需迁移。
- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹。复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI、MSIX 或 APPX。
- 当前发布二进制未进行 Authenticode 签名。
