# QuickPaste v0.19.0 验证报告

验证日期：2026-07-22

本报告记录 v0.19.0 发布候选的本地自动化、安全审计、界面检查和产物验证证据。本次不使用 Computer Use；真实应用完整交互矩阵、真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 规划范围

- 快速面板在既有 44px/40px 行高内将有效摘要扩展为最多两行。
- 管理页把重复标题与正文合并为单一两行摘要，并保留搜索高亮、徽标、键盘选择和行内操作。
- 不修改数据库 schema、捕获协议、粘贴协议、分页协议、依赖或系统权限。

## 自动化与安全审计

发布门禁执行完成后回填最终测试数量、npm/RustSec 审计和 Windows 构建结果。

## 界面检查

Playwright CLI 检查完成后回填标准/紧凑窗口行高、可见条目数、两行截断、管理页去重、键盘焦点及控制台结果。

## 发布产物

Windows 构建完成后回填主程序、NSIS 安装包与绿色版 ZIP 的精确字节数和 SHA-256，并记录 PE machine、版本、签名状态及 GitHub 回下载校验。

## Pending real-machine

- Windows 10/11 覆盖安装、卸载、保留 `data` 与重装。
- 100%–250% DPI、多显示器负坐标、不同任务栏位置和辅助技术组合。
- Chrome/WebView2、Office、记事本及远程剪贴板工具组合下的长期竞争与完整富文本兼容矩阵。
- 500 次暖唤起、10,000 次外部普通目标粘贴、100,000 次独立捕获与长时间监听。
- 管理员目标、UAC 取消、WebView2 缺失和在线 Bootstrapper 路径。
