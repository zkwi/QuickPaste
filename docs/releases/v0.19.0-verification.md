# QuickPaste v0.19.0 验证报告

验证日期：2026-07-22

本报告记录 v0.19.0 发布候选的本地自动化、安全审计、界面检查和产物验证证据。本次不使用 Computer Use；真实应用完整交互矩阵、真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 规划范围

- 快速面板在既有 44px/40px 行高内将有效摘要扩展为最多两行。
- 管理页把重复标题与正文合并为单一两行摘要，并保留搜索高亮、徽标、键盘选择和行内操作。
- 不修改数据库 schema、捕获协议、粘贴协议、分页协议、依赖或系统权限。

## 自动化与安全审计

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，46 项治理测试通过。
- 前端测试：34 个测试文件、724 项测试通过；TypeScript 检查与 Vite 生产构建通过。
- Rust 格式、Clippy（`-D warnings`）及测试：317 项测试通过。
- `npm audit --omit=dev --audit-level=high`（官方 npm registry）：通过，0 个漏洞。
- `cargo audit --file src-tauri/Cargo.lock`：退出码为 0，没有阻断性漏洞；报告 17 条允许的维护状态或 soundness 警告，来自 Tauri 的 GTK3/UNIC 间接依赖。QuickPaste 当前只交付 Windows，这些依赖不进入本次 Windows 可执行文件；警告仍由每周 RustSec 工作流持续监控。
- `npm run build:windows`：通过，只生成 v0.19.0 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 界面检查

- 使用 Playwright CLI 在 800×580 浅色窗口检查标准模式：行高为 44px，9 条构造记录完整可见；长摘要高度为 36px、实际内容高度为 54px，并按两行截断。
- 使用 Playwright CLI 在 640×440 深色窗口检查紧凑模式：行高为 40px，7 条记录完整可见（门槛为 6 条），两行摘要未撑高条目。
- 管理页 9 条记录全部使用统一摘要；旧标题节点和重复正文段落均为 0，构造的重复标题只出现一次，独立标题与正文合并展示。
- 三个页面均验证键盘焦点移动、行内操作保留且无横向/纵向页面溢出；控制台 0 个错误、0 个警告。
