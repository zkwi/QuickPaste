# QuickPaste v0.14.0 验证报告

验证日期：2026-07-20

本报告记录 v0.14.0 发布候选在本地 Windows 构建环境中的自动化、界面和产物验证证据。真实设备长时间运行及完整安装生命周期仍单独列为待验证项。

## 自动化验证

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过。
- 前端测试：31 个测试文件、674 项测试通过。
- TypeScript 检查与 Vite 生产构建：通过。
- Rust 格式、Clippy（`-D warnings`）及测试：292 项测试通过。
- `npm run build:windows`：通过，只生成 v0.14.0 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 关键回归覆盖

- 正常运行把 `QuickPaste.exe` 映射到同级 `data`，验收模式显式 profile 仍保持优先且隔离。
- 原生“打开数据目录”命令只向前端返回成功状态，不返回用户绝对路径。
- 存储管理页在中英文中说明必须完全退出后复制整个 `data`，并提供可操作入口与失败反馈。
- 首次引导主动居中、失焦不隐藏；托盘、快捷键或重复启动应用时优先恢复仍未完成的引导。
- 快速面板、管理页和引导页标题区域使用原生深层拖动，按钮、链接和表单控件仍保持交互。

## 界面验证

使用真实 Chromium 页面检查了设置与存储区域：

- 800×580 浅色主题与 640×440 深色主题。
- 数据位置说明、打开目录按钮、数据库大小和数据条数在标准宽度横向对齐，在紧凑宽度自然改为纵向布局。
- 存储区所有后代元素均未出现横向内容溢出；按钮边界保持在提示卡片内。
- 页面可访问快照包含完整中文说明与操作名称；浏览器控制台无 warning 或 error。
