# QuickPaste v0.10.0 发布说明

v0.10.0 重点提高高频页面的信息密度，并把更新入口从设置项提升为可直接操作的右下角通知。本版本以 [GitHub Pre-release](https://github.com/zkwi/QuickPaste/releases/tag/v0.10.0) 分发；真实机长循环仍保留为待验证项，因此不表述为稳定版已全面验收。

## 更紧凑的快速面板

- 来源应用和时间改为上下排列的固定宽度信息轨道，正文可使用更多横向空间。
- 列表继续保留清晰的类型图标、键盘序号和选中状态，不增加额外说明卡片。
- 图片详情把图片与 OCR 文字放在同一内容区；空间不足时自动回落为单列。

## 更精简的管理与统计

- 管理页移除顶部统计卡片、来源精确筛选和固定状态筛选，只保留全文搜索与内容类型。
- 批量工具栏仅在选中记录后出现，空闲状态不再显示一排不可用按钮。
- 来源应用和时间在管理列表中上下排列，统一收紧行高、缩略图和操作间距。
- 设置页的存储概览只保留数据库大小与数据条数；大小按 MB 显示，不再堆叠 WAL、SHM、逻辑载荷和保留策略明细。

## 可直接操作的更新通知

- 自动检查发现新版本后，在窗口右下角显示更新通知，可直接点击“下载安装”。
- 通知会显示版本和安装包大小，并在下载期间展示进度；用户也可以随时关闭。
- 下载后仍会核对 GitHub 提供的 SHA-256 摘要，并在安装前确保剪贴板历史已经安全落盘。

## 下载与验证

- Windows x64 当前用户 NSIS：[`QuickPaste_0.10.0_x64-setup.exe`](https://github.com/zkwi/QuickPaste/releases/download/v0.10.0/QuickPaste_0.10.0_x64-setup.exe)。
- 自动化、真实页面、候选安装包摘要与未完成证据记录见 [v0.10.0 验证报告](https://github.com/zkwi/QuickPaste/blob/main/docs/releases/v0.10.0-verification.md)。

安装程序仍使用在线 WebView2 Bootstrapper。目标机器缺少 WebView2 Runtime 时，安装过程需要联网；离线安装尚未作为发布能力验收。

## 尚待真实机长循环

- 升级安装、卸载、重启后自动启动与托盘单实例闭环。
- 多显示器、不同 DPI、管理员目标窗口及 Windows 10/11 的持续人工回归。
- WebView2 缺失机器上的在线 Bootstrapper 安装流程。

以上项目仍为 `pending real-machine`，因此 v0.10.0 保持 Pre-release。
