# QuickPaste v0.16.0 验证报告

验证日期：2026-07-21

本报告记录 v0.16.0 发布候选的本地自动化和产物验证证据。真实设备长时间运行、完整安装生命周期和完整浏览器视觉矩阵继续单独列为 `pending real-machine`。

## 自动化验证

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，41 项治理测试通过。
- 前端测试：34 个测试文件、710 项测试通过；TypeScript 检查与 Vite 生产构建通过。
- Rust 格式、Clippy（`-D warnings`）及测试：315 项测试通过。
- 10,000 条历史 release 合成基准：短查询 p95 20.272ms，门槛 50ms；基准数据准备约 212.5 秒。最终发布阶段按维护者要求不重复运行长基准。
- `npm run build:windows`：通过，只生成 v0.16.0 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 界面检查范围

- 自动化覆盖快速面板、资料库、设置、更新失败恢复、键盘焦点和中文输入法组合态。
- 本轮只完成浏览器快速面板 HTTP 200 与渲染快照；完整浅色/深色、紧凑/标准、管理/设置和 reduced-motion 真实页面矩阵未完成，保留为 `pending real-machine`。

- 绿色版只包含 `QuickPaste.exe` 和四份第三方许可文件，其中的可执行文件与 release 构建目录摘要一致。
