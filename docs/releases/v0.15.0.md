# QuickPaste v0.15.0 发布说明

v0.15.0 面向中文用户的高频剪贴板工作流，加入永久片段快速检索、来源建议、本地二维码识别和更清晰的固定语义，同时尊重 Windows 应用设置的敏感剪贴板历史标记。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机长循环仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- 搜索框输入 `;` 或 `；` 可快速调用永久片段；输入 `@` 可从来源应用建议中选择微信、飞书、WPS、浏览器等来源，并继续叠加正文关键词。
- 图片预览中的本地 OCR 文字可直接复制或粘贴；二维码完全在本机识别，普通文本可复制，安全的 HTTP/HTTPS 链接可直接打开。
- Windows 剪贴板明确声明不应进入历史时，QuickPaste 会在读取正文前跳过；仅禁止云剪贴板上传的内容仍可保留在本机。
- “面板常驻”与“固定内容”使用不同图标、文案和 Tooltip；设置页会提示常见粘贴快捷键冲突。

## 数据与兼容性

- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹；本版本不改变历史数据结构的兼容方式。
- 复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI。

## 尚待真实机验证

- Windows 10/11 的覆盖安装、卸载、保留 `data` 与重装。
- 100%–250% DPI、多显示器负坐标和不同任务栏位置。
- 长时间剪贴板监听、敏感标记来源应用、管理员目标回贴和 WebView2 缺失路径。
