以下项目保持 **pending real-machine**，必须绑定最终不可变提交和候选 SHA-256 后执行：

- 50 次预热 + 500 次快捷键到首帧采样，p95 ≤ 120 ms。
- 由独立普通权限目标实际读取 10,000 次粘贴内容，成功率 ≥ 99.5%。
- 由独立 writer/expected-ID ledger 执行 100,000 次捕获并与数据库/事件对账，漏记、异常重复和自身回流合计 < 0.1%。
- 100%、125%、150%、175%、200%、225%、250% DPI，多显示器不同缩放、负坐标和四任务栏边缘无裁切错位。
- Explorer/Office/浏览器/管理员目标、中文输入法、Windows 已安装语言包 OCR、安装/覆盖安装/卸载，以及真实机网络抽查。
- 使用可信 NSIS 解包工具枚举完整归档；第三方 npm、Rust、NSIS/LZMA 与 SQLite 清单已经生成、打包并通过锁文件一致性检查。
