# QuickPaste v0.17.0 发布说明

v0.17.0 改进快捷面板在文本插入点附近的定位可靠性，并把鼠标悬停反馈从快捷键提示改为可直接辨认的长文本与大图预览。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机长循环仍保留为待验证项，因此不表述为已经全面验收。

## 主要变化

- Windows 快捷面板优先使用系统辅助功能报告的插入光标矩形；当应用只返回零高度光标点时，按当前显示器 DPI 补足合理行高，再受工作区边界约束定位。
- 鼠标在文本或图片记录上停留 200ms 后显示浮动预览：文本在标准窗口最多八行、紧凑窗口最多六行，图片使用完整图片载荷并保持比例。
- 悬停预览不改变键盘选中项和焦点；超长正文最多挂载 4096 个 UTF-16 单元并追加省略号，避免大文本拖慢 WebView。
- Alt 数字徽标只显示在键盘当前选中记录上，鼠标悬停不再弹出主记录的快捷键提示；`aria-keyshortcuts` 与键盘快速粘贴保持不变。

## 数据与兼容性

- 本版本不改变 SQLite schema、数据目录或历史保留规则，无需迁移。
- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹。复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI。

## 尚待真实机验证

- Windows 10/11 的覆盖安装、卸载、保留 `data` 与重装。
- 100%–250% DPI、多显示器负坐标、不同任务栏位置，以及更多应用对 MSAA 插入光标位置的实际支持情况。
- 500 次暖唤起、10,000 次外部普通目标粘贴、100,000 次独立捕获与长时间监听。
- 管理员目标、UAC 取消、WebView2 缺失和在线 Bootstrapper 路径。

以上项目保持为 `pending real-machine`，不作为已完成验证陈述。
