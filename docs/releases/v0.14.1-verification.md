# QuickPaste v0.14.1 验证报告

验证日期：2026-07-20

本报告记录 v0.14.1 发布候选的本地自动化、人工界面和产物验证证据。真实设备长时间运行及完整安装生命周期仍单独列为待验证项。

## 自动化验证

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过。
- 前端测试：31 个测试文件、674 项测试通过。
- TypeScript 检查与 Vite 生产构建：通过。
- Rust 格式、Clippy（`-D warnings`）及测试：293 项测试通过。
- `npm run build:windows`：通过，只生成 v0.14.1 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 关键回归覆盖

- WebView2 短暂失焦但 QuickPaste 顶层窗口仍在前台时，快速面板不自动隐藏。
- 顶层 QuickPaste 窗口确实失去前台状态时，未固定快速面板仍自动隐藏。
- 实际无边框测试实例保持原有视觉样式，用户复测确认顶部区域可拖动且窗口不会消失。
