# QuickPaste v0.19.1 验证报告

验证日期：2026-07-22

本报告记录 v0.19.1 拼音索引命中摘要修复的自动化、界面和候选产物验证证据。本次不使用 Computer Use；真实应用完整交互矩阵、真实设备长时间运行、完整安装生命周期和完整 DPI 矩阵继续单独列为 `pending real-machine`。

## 缺陷回归

- 构造原生历史摘要：标题为正文前缀，正文长度足以占用两行，查询词 `xiamen` 仅通过 `matchSource: index` 命中。
- 修复前回归测试稳定收到短标题而不是正文摘要；修复后快速面板收到完整有界正文摘要，并继续显示“索引命中”提示。
- 修复只让仅索引命中复用已有普通摘要规则，不增加完整载荷读取、数据库查询、剪贴板访问或网络访问。

## 自动化与安全审计

- `npm ci`：通过，共安装 140 个锁定包。
- 治理、版本、仓库链接、隐私、运行时边界和第三方许可检查：通过，46 项治理测试通过。
- 前端测试：34 个测试文件、725 项测试通过；TypeScript 检查与 Vite 生产构建通过。
- Rust 格式、Clippy（`-D warnings`）及测试：317 项测试通过。
- `npm audit --omit=dev --audit-level=high`（官方 npm registry）：通过，0 个漏洞。
- `cargo audit --file src-tauri/Cargo.lock`：退出码为 0，没有阻断性漏洞；报告 17 条允许的维护状态或 soundness 警告，来自 Tauri 的 GTK3/UNIC 间接依赖。QuickPaste 当前只交付 Windows，这些依赖不进入本次 Windows 可执行文件；警告仍由每周 RustSec 工作流持续监控。
- `npm run build:windows`：通过，只生成 v0.19.1 x64 NSIS 安装包，未生成 MSI、MSIX 或 APPX。

## 界面检查

- 使用 Playwright CLI 在 800×580 浅色标准模式构造 `xiamen` 拼音索引记录；可见摘要包含完整有界正文而不是短标题。
- 摘要可见高度为 36px、行高为 18px，`-webkit-line-clamp: 2` 生效；记录行高仍为 44px，没有因两行正文而增长。
- “拼音命中”徽标保持可见；该布局与原生模式的“索引命中”徽标共用同一组件和样式。控制台 0 个错误、0 个警告。
