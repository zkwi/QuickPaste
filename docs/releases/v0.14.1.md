# QuickPaste v0.14.1 发布说明

v0.14.1 修复无边框快速面板从顶部拖动时可能立即消失的问题。本版本以标准 GitHub Release 分发，不标记为 Pre-release；真实机长循环仍保留为待验证项，因此不表述为已经全面验收。

## 修复内容

- 顶部区域继续使用原有无边框、透明圆角样式，无需增加 Windows 系统标题栏。
- 开始原生窗口拖动时，WebView2 的短暂失焦不再触发快速面板自动隐藏。
- 用户真正切换到其他应用时，未固定的快速面板仍按原有逻辑自动收起。

## 数据与兼容性

- 数据库、WAL 和 SHM 继续保存在 `QuickPaste.exe` 同级的 `data` 文件夹，本版本不迁移开发阶段旧目录。
- 复制或迁移前必须从托盘完全退出 QuickPaste，避免遗漏仍在 WAL 中的数据。
- 安装包仍只提供当前用户范围的 Windows x64 NSIS，不生成 MSI。

## 尚待真实机验证

- Windows 10/11 的覆盖安装、卸载、保留 `data` 与重装。
- 100%–250% DPI、多显示器负坐标和不同任务栏位置。
- 长时间剪贴板监听、管理员目标回贴和 WebView2 缺失路径。

以上项目保持为 `pending real-machine`，不作为已完成验证陈述。
