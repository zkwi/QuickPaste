# 0001：Windows 优先的 Tauri 模块化单体

- 状态：Accepted
- 日期：2026-07-19

## 背景

QuickPaste 是高频常驻的 Windows 剪贴板工具，需要较小安装体积、Windows API、透明窗口、全局快捷键和可快速迭代的 UI。当前由个人项目维护，没有跨平台交付或独立服务需求。

## 决策

使用 Vue 3 + TypeScript 实现交互，使用 Tauri 2 + Rust 承载 Windows、SQLite 和输入能力，并保持一个可部署单体。前端按 `domain`、`platform`、UI 编排分层；Rust 只在代码规模或测试边界产生明确收益时拆模块。

## 结果

- 可以使用 Web 技术快速打磨 UI，同时避免 Electron 级别的额外运行时体积。
- 原生能力和 UI 之间需要维护明确 IPC 契约及失败降级。
- Windows 是当前唯一验收平台；保留平台适配层不等于提前实现其他端。
- 大文件不是单独拆分理由，重复、边界泄漏或难以测试才是拆分触发条件。

## 重新评估条件

只有出现真实的第二平台交付、插件隔离、独立同步服务或单体测试/构建成本不可接受时，才重新评估该决策。
