# 测试与验收

自动化测试负责稳定业务规则和系统契约；真实 Windows 验收负责 DPI、焦点、输入法、管理员权限、WebView2 和安装器等无法由 happy-dom 或纯 Rust 单元测试完整覆盖的行为。

## 自动化命令

| 命令 | 覆盖范围 |
| --- | --- |
| `npm run test:governance` | 仓库检查脚本自身的单元测试 |
| `npm run check:versions` | 版本、NSIS、工具链及自定义标题栏 Tauri 权限契约 |
| `npm run check:repository` | 生成目录、敏感文件、单文件体积和 Markdown 相对链接 |
| `npm run check:privacy` | 拦截个人路径、邮箱、Token、数据库、日志、截图和构建产物 |
| `node --test scripts/acceptance/acceptance.test.mjs` | 验收准备器、阈值边界、JSON schema 与 PowerShell 语法；不会运行真实机长循环 |
| `npm test` | domain、platform 和 Vue 交互回归 |
| `npm run build` | TypeScript 类型检查和 Vite 生产构建 |
| `npm run check:rust` | Rustfmt、Clippy 和 Rust 单元测试 |
| `npm run check` | 以上全部门禁 |

修改行为时先增加会失败的最小回归测试。测试数据只能使用构造内容，不得复制真实聊天、凭据、本机路径或剪贴板数据库。

## 证据分级

| 级别 | 能证明什么 | 不能证明什么 |
| --- | --- | --- |
| **automated proof** | 单元/集成测试中的规则、序列化白名单、阈值边界、临时 profile 安全约束 | 真实 Windows 的焦点、目标收到内容、合成器呈现、长时间剪贴板吞吐 |
| **synthetic benchmark** | 固定种子和受控环境中的 10k SQLite 查询/构建性能 | 最终用户机器的快捷键首帧、普通目标粘贴成功率或 100k 捕获完整性 |
| **pending real-machine** | 表示对应真实机证据尚未完成或不完整 | 不能表述为已达标、通过或由自动化间接证明 |

真实机验收使用 [验收脚手架与协议](../scripts/acceptance/README.md)。每个场景必须由显式 `-OptIn` 启动独立临时 profile；生产默认不得创建 metrics。仅同时提供 `--acceptance-metrics` 与有效 `QUICKPASTE_ACCEPTANCE_PROFILE` 时，候选才可在固定 profile 相对路径 `acceptance/metrics-v1.json` 原子写入内容无关的有界快照；缺少 profile 时必须在 Tauri setup 前失败，不能回落真实 app-data。其 schema 为 [metrics-v1.schema.json](../scripts/acceptance/metrics-v1.schema.json)。

暖唤起、普通目标粘贴、捕获完整性和 DPI 分别需要 50+500、10,000、100,000 和 1.0–2.5 七档混合显示器记录。脚手架自测或 synthetic benchmark 通过时，这些项目仍保持 **pending real-machine**；普通粘贴必须由外部普通权限目标读取实际收到的标记，捕获必须由独立 writer/expected-ID ledger 与最终 DB/事件对账。

## 变更对应的最低验证

| 变更范围 | 自动化 | 额外人工验证 |
| --- | --- | --- |
| domain 纯规则 | 对应 Vitest + `npm run typecheck` | 通常不需要 |
| Vue UI、样式、焦点 | 对应 App 测试 + `npm run build` | 紧凑/标准窗口、浅色/深色、键盘 |
| IPC、窗口、快捷键 | platform 测试 + Rust 定向测试 | 真实 Tauri 窗口、多显示器、DPI |
| SQLite、历史迁移 | domain/platform + Rust 数据库测试 | 重启恢复、损坏数据只读保护 |
| 管理员粘贴 | Rust 安全授权测试 | 普通目标、管理员目标、UAC 取消 |
| 安装配置 | `npm run check` + `npm run build:windows` | 安装、升级、卸载、SHA-256、未签名提示与 WebView2 |

## Windows 人工矩阵

发布候选至少在 Windows 10 和 Windows 11 的 x64 标准用户账户验证；涉及定位或透明外壳时，还需覆盖 100%、125%、150%、175%、200%、225%、250% DPI 及双显示器不同缩放、负坐标与四个任务栏边缘。

### 快速面板

- 在记事本、微信等可编辑文本框中唤起，面板优先位于插入点右下方且不遮挡插入点。
- 无可用文本插入点时回退到鼠标附近；靠近四角、任务栏和副屏时仍完全位于工作区内。
- 小工作区会自适应缩小，边框、圆角、阴影和透明外壳没有矩形漏底或双层描边。
- `800 × 580` 标准窗口至少完整显示 8 条双行记录；`640 × 440` 紧凑窗口至少完整显示 6 条，标题、摘要、来源和时间均不重叠。
- 浅色与深色主题下搜索框、筛选项和行内操作仍清晰；标准行高不低于 44px、紧凑行高不低于 40px，搜索框不低于 32px。
- 新会话清空上次搜索、筛选、预览、滚动位置和临时反馈。
- 冷启动和历史重载期间复制的多条内容在历史就绪后完整合并，顺序正确且不覆盖已有历史。
- 鼠标悬停不改变键盘选择；方向键、Home/End、Enter、Escape、Alt + 数字行为与可见状态一致。

### 历史与搜索

- 文本、链接、代码和图片捕获后顺序正确；重复内容移到首位但不重复，固定状态保留。
- 搜索忽略大小写并兼容全角/半角；高亮保留原字符，不把仅由隐藏关键词命中的中文伪装成可见命中。
- 长内容摘要围绕首个命中显示；清空搜索后原文完整恢复。
- 中文输入法组合态中，Enter 和方向键不会误触发粘贴或导航。
- 重启后 SQLite 历史和图片恢复；加载失败时界面只读，不写入空历史。

### 粘贴与权限

- 普通窗口 Enter、双击和 Alt + 数字均粘贴正确条目。
- 目标失效、焦点恢复失败或修饰键未释放时只复制，并给出可执行的手动粘贴提示。
- 管理员目标只通过一次性 helper；UAC 取消、超时、重复发起、目标句柄变化或剪贴板序列变化时不得提升主进程、堆积授权流程或误贴到其他窗口。
- 直接以管理员身份启动主程序时必须提示并退出，历史数据库和常驻界面不得在高完整性下运行。
- 固定面板的连续粘贴和普通一次性面板的隐藏行为分别符合当前产品约定。
- 文本和图片两条链路均验证，快速连续按键不会重复执行同一次操作。

### 设置、隐私与可访问性

- 开机启动、全局快捷键、暂停记录、排除应用、保留期限和语言设置重启后保持一致。
- 清理历史保留固定项；破坏性操作有确认、取消和焦点恢复。
- 浅色、深色、Windows 高对比度和 reduced-motion 下，文本、焦点环和搜索高亮仍清晰。
- 键盘可完成主流程；读屏能识别搜索结果数量、当前选择、按钮目标和错误状态。
- 敏感应用排除和屏幕捕获排除按支持范围工作，同时不把尽力防护表述为绝对保证。
- 默认或关闭“屏幕捕获保护”时，分别使用 `Win + Shift + S`、微信截图及窗口/区域捕获确认 QuickPaste 内容可见；隐藏后再次唤起仍可捕获。
- 主动开启“屏幕捕获保护”后，在 Windows 支持的捕获路径中确认窗口隐藏或显示空白；再次关闭后无需重启即可恢复截图，重启后仍保持用户最后一次明确选择。

### 原生窗口

- 管理器按钮可从普通状态最大化并再次还原，不出现“窗口操作未完成”。
- 标题栏双击或 Windows Snap 进入最大化后，自定义按钮仍可还原。
- 最大化时内容铺满工作区，无透明矩形边缘、悬空圆角或居中的固定尺寸小面板；按钮图标和读屏文案切换为“还原”。
- 快速连续点击窗口按钮只执行一次；窗口模式切换期间控制按钮保持禁用。

### 安装与生命周期

- 标准用户完成当前用户范围安装、覆盖安装和卸载，不要求主程序常驻管理员权限。
- 机器已有/缺少 WebView2 两种情况分别验证；缺少 Runtime 时确认在线 Bootstrapper 的失败提示。
- 安装后启动、单实例、托盘、关闭隐藏、开机静默启动和卸载后重装正常。
- 卸载不意外删除用户历史；彻底删除数据的说明与 [SECURITY.md](../SECURITY.md) 一致。

## 验收记录

人工验收应记录版本、提交标识、候选程序 SHA-256、Windows build、DPI/显示器、安装方式、结果分类和脱敏截图。长循环与 DPI 结果使用脚手架生成的 `result.json`，先通过 JSON schema 和 `validate-result` 算术校验；未测项目明确保留 **pending real-machine**。截图、metrics、数据库、日志与账本不得提交，截图也不得包含真实剪贴板、账户名、聊天记录或本机隐私路径。
